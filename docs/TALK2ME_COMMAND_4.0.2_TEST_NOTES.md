# Talk2Me Command 4.0.2 — Restored Inquiry Service Actions

- New Inquiry again displays its own customer search.
- `client_id` links from Customer Workspace now select the correct customer.
- The selected customer is no longer cleared while the inquiry opens.
- Selecting a customer displays Add Mobile Line, Add Fixed Service and Open Customer.
- Staff actions continue through approval; owner and manager actions use their direct capture routes.
- Fixed-account inquiries display the same service actions.

No database migration is required.
