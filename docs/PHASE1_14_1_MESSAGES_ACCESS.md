# Talk2Me CRM v1.14.1 - Messages access

- Adds a visible Messages & Tasks card to Back Office.
- Adds a labelled Messages link with unread badge in the top navigation.
- Owner/admin/manager send messages from Back Office > Messages & Tasks > Create Task.
- Staff receive messages through the Messages bell/list and by email to their login email address.
- No database migration or npm install required after v1.14.0.
