# Talk2Me Command 4.0.6

- Replaces the browser icon with a high-visibility red circular Talk2Me mark.
- Includes 16, 32, 48 and 64 px favicon sizes plus 192 and 512 px app icons.
- Adds versioned icon URLs to force browsers to refresh their favicon cache.
- Includes all Version 4.0.5 workflow corrections.

No SQL migration and no npm install are required.
