# Talk2Me Command 4.0.1 — Test Notes

Version 4.0.1 refines the operational workspace introduced in 4.0.0.

- Talk2Me red and Chatz blue visual identity.
- Red menu button and prominent red-outlined universal search.
- Mobile and Fixed sidebar links removed; universal search covers both.
- New Inquiry moved directly below My Workspace.
- Large greeting and clock removed; a five-second login greeting replaces them.
- Quick actions and operational queues now appear first.
- Summary totals moved to the bottom of the workspace.
- Existing routes, permissions, database records and CRM functions are unchanged.

Deploy as a full replacement of application files, run NPM Install, restart Node.js and hard-refresh the browser. No database migration is required.
