# Phase 1.9 - Favicon Fix

This patch updates the browser tab icon to use the new Talk2Me logo assets.

## Install

1. Upload/extract over `public_html/talk2me`.
2. Restart the Node.js app.
3. Hard refresh the browser.

## Browser cache note

Favicons are aggressively cached. If the old icon still shows, open in an incognito window or hard refresh.
