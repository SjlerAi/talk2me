# Talk2Me CRM v3.2.1 — streamlined Client Administration

The Client Administration list now focuses only on database administration.

- Owners and managers can assign or unassign the responsible staff member.
- Owners and managers have one Edit Line action per row.
- Staff can see the assignment but the row remains view-only.
- Customer 360 is reached through the main customer search.
- New inquiries are started through the inquiry workflow.
- Proposed changes remain in the staff workflow and Approvals area.

No database migration is required when upgrading from v3.2.0.
