# Talk2Me CRM v3.2.5 — account additions

- Add Mobile Line and Add Fixed Service now appear in the main Fixed Customer 360 action row.
- The duplicate Add Fixed Service button was removed from the fixed-services table.
- A mobile line can now be started from a fixed-only customer account.
- Owners and managers add directly; staff submit the new mobile or fixed service for approval.
- New services and lines remain linked to the existing canonical unique account.
- No database migration is required.
