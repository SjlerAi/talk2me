# Talk2Me CRM v1.13.0 - Birthday and Upgrade Dashboard

Adds morning dashboard panels for authorised owner/admin/manager users:

- Birthdays today
- Birthdays tomorrow
- Upgrades due today
- Upgrades due during the following 7 days
- Client name, contact details, handset, package, account number and exact upgrade date

No database migration or NPM install is required.

Deployment:
1. Extract this flat ZIP into `public_html/talk2me` and overwrite existing files.
2. Set `APP_VERSION=1.13.0`.
3. Restart the Node.js app.
4. Hard-refresh the browser.
