# Talk2Me CRM Phase 1.16.0

## Added

- Friendly branded Talk2Me task and notification emails.
- Email subject: `🔔 New Talk2Me task for you` (or notification equivalent).
- Red button opens the exact task or notification in the CRM.
- Live in-app popup appears for unread messages while staff are logged in.
- Popup includes sender, priority, due date, message preview and an Open button.
- Polls every 12 seconds without refreshing the page.

## Install

1. Extract the flat ZIP over `public_html/talk2me`.
2. Set `APP_VERSION=1.16.0`.
3. Save and restart the Node.js app.
4. Hard refresh the browser.

No SQL migration or NPM install is required.
