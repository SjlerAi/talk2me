# Talk2Me CRM v1.20.1 – Form Privacy & Reset

- Removed test login credentials from the login screen.
- Login username and password are cleared whenever the login page loads or is restored from browser history.
- Quick Add New Contact fields are cleared each time the panel opens.
- Only the currently searched telephone number is carried into the quick-contact form.
- New inquiry client/contact fields start blank and are cleared when a fresh inquiry page is opened.
- Added autocomplete protections to reduce browser reuse of previous contact details.
